urldecode (str)
Decodes a URL encoded string '+' are decoded to spaces, Hexidecimals are decoded to their alpha-numeric equivalent

Parameters:

  • str The url encoded string to decode

Usage:

    local example = "this+is+a+%C2%A3test+%3Cto+encode%3E"
local decoded = qs.urlencode(example)

Returns:

    The decoded string
urlencode (str)
Encodes a string to be used in the query part of a url. Spaces are encoded to plus (+) signs. Non alpha-numeric characters are encoded to their hexadecimal equivalent.

Parameters:

  • str The string to encode

Usage:

    local example = "this is a £test "
local encoded = qs.urlencode(example)

Returns:

    The url encoded string
parse (str, sep, eq)
Encodes a string to be used in the query part of a url. Spaces are encoded to plus (+) signs. Non alpha-numeric characters are encoded to their hexadecimal equivalent.

Parameters:

  • str The string to encode
  • sep Character to separate parameters. Default is &
  • eq Character for value assignation. Default is =

Usage:

    local example = "this is a £test "
local encoded = qs.urlencode(example)

Returns:

    The url encoded string
